Habilidades

Studio Code – Editor de código leve e poderoso, com suporte a extensões e recursos para produtividade.
Tailwind CSS – Framework de CSS que permite criar estilos de forma rápida usando classes pré-definidas.
React – Biblioteca JavaScript para criar interfaces modernas, rápidas e reutilizáveis.
TypeScript – Versão aprimorada do JavaScript com tipagem estática, que ajuda a evitar erros no desenvolvimento.
JavaScript – Adiciona interatividade e dinamismo ao site, como animações, botões e validações.
GitHub – Plataforma online para hospedar projetos, compartilhar código e colaborar com outros desenvolvedores.
Git – Sistema de controle de versão para acompanhar mudanças no código e facilitar trabalho em equipe.
HTML – Linguagem de marcação usada para estruturar o conteúdo de páginas da web.
CSS – Estiliza as páginas, definindo cores, fontes, espaçamentos e deixando o site visualmente atrativo.
Figma – Ferramenta de design usada para criar interfaces e protótipos de sites e aplicativos, facilitando a colaboração entre designers e desenvolvedores.
Next – Framework baseado em React que otimiza o desempenho do site, oferecendo rotas dinâmicas, SEO aprimorado e renderização rápida.
PostgreSQL – Banco de dados relacional robusto e seguro, usado para armazenar e gerenciar informações de forma estruturada.
